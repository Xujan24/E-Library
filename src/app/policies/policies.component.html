Terms & Conditions

1. Acceptance of the Terms of Use

You will need to provide a valid email address in order to use the system. An email with the verification link will be sent to the provided email address. Once you verified your email, you can delete your account at any time and your email will be permanently removed from our database. We only ask you to provide your genuine email address for verification purpose. And your email address won't be used for any other proposes.

Data Storage

E-library uses Google Cloud Firestore, a cloud based realtime Database, to store user's data (browsed items, history and saved items). It is to note that all the data will be permanently deleted at 11:59 pm UTC, everyday or two. This will include the user information (email) and all other user contents (borrowed items, saved items and history). Further information about Cloud Firestore can be found here.

2. Technologies and Services

E-library is built with Angular 8 and Google Firebase. By using E-library portal, you automatically agree their respected terms and conditions. The following is a complete list of external libraries and services used in this project.

AngularFire

AngularFire is the official library for Firebase and angular. It offers various services like Firebase Cloud Firestore, Firebase Authentication, Firebase Storage and so on and synchronize data in real time. A complete documentation for AngularFire is available here.

Google Books

Google Books, previously known as Google Book Search and Google Print, is a service from Google Inc. that searches the full text of books and magazines that Google has scanned. The terms of service for Google Books is availabe here.

Font Awesome

Font Awesome is one of the world's most popular icon set and toolkit. It offers a free version and a pro version. This project uses free version of Font Awesome.

Single Element CSS Spinners

Single Element CSS Spinners is a collection of various spinners built with pure CSS, created by @lukehaas. You can check the project on Github and a live demo here.
